🚀 TrackShot — FPS Aim Trainer (Inspired by Valorant)

TrackShot is a browser-based aim training tool designed to improve accuracy, reaction time, and tracking skills for FPS players. Built using modern web technologies, it simulates core aiming mechanics found in competitive shooters.

🎯 Features:
🔹 Multiple Training Modes
      Static Click
      Tracking
      Flick Shot
      Speed Mode
🔹 Real-Time Performance Tracking
      Hits / Misses
      Accuracy %
      Reaction Time
🔹 Interactive Dashboard
      Session history
      Performance graphs
      Insights based on gameplay
🔹 Sensitivity Control
      Adjustable in-game sensitivity
      Designed to mimic FPS aim feel
🔹 Crosshair Customization
      Multiple crosshair styles
      Live preview system
🔹 Backend Integration
      Stores session data
      Retrieves history dynamically

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Graphics: Canvas / Three.js
Backend: Python (Flask)
Database: SQLite


📊 Use Case
This project is useful for:

1) FPS players wanting to improve aim
2) Learning game mechanics in browser
3) Demonstrating full-stack development skills
